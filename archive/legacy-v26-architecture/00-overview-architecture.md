
# 00-overview-architecture.md

### Purpose This wiki is an Obsidian-based, LLM-friendly knowledge system for humanitarian and DRR work focused on area-based approaches. It is designed to:
- Capture and structure evidence from sources. - Preserve contradictions and tensions explicitly. 
- Translate knowledge into operational decisions, tools, and field instruments. 
- Support agents and humans in designing, running, and adapting ABA programmes. 
- Core Principles - Evidence traceability via `extracted/` and required `contradicts:`. - Separation of raw evidence from synthesis. - Layered structure: Sources → Concepts → Frameworks → Tools → Instruments → Lifecycle → Sector → Coordination → MEL → Transition → Patterns → Risks. - Independence enforcement for concepts and patterns. - Agents infer application from context (especially lifecycle), rath``
