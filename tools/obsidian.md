# Tool: Obsidian

## Purpose

Obsidian is the markdown knowledge base application used to view, navigate, and edit this wiki as a human operator.

## Used For

- Browsing the wiki with `[[wikilink]]` navigation
- Searching across vault files
- Reviewing memory, decisions, and logs
- Editing templates, workflows, and prompts
- Visual graph of note connections (optional)

## Safety Rules

- This wiki is designed to work without Obsidian plugins — do not require plugins for agent navigation.
- Do not enable community plugins that auto-sync, auto-publish, or send data externally without reviewing what they transmit.
- Do not store secrets in any file visible in the vault.

## Notes

- All files in this vault are standard markdown. They can be opened in any text editor.
- Obsidian `[[wikilinks]]` resolve relative to the vault root by default.
- This vault is not configured with an Obsidian `.obsidian/` folder intentionally — open the folder as a vault in Obsidian to auto-generate it.
- Graph view is useful for exploring connections between memory categories and workflows.
