<a id="trigger"></a>
## Trigger Conditions

Immediately after a disaster, when pursuing the full Characteristics set is unrealistic. *"A week after the flood — what resilience goals are realistic?"*

<a id="decision-logic"></a>
## Decision Logic

**Rule:** select a **small set of key or minimum Characteristics** to serve as recovery **milestones**, rather than attempting comprehensive coverage. Pairs with the milestones tool (T-measure-resilience-progress) for tracking.

<a id="escalation"></a>
## Escalation

Escalate to the response/recovery lead when the minimum set cannot be agreed, or when life-safety needs conflict with the chosen milestones.
