<a id="definition"></a>
## Definition

The **enabling environment** is the set of policy, institutional, and socio-economic factors — largely outside a community's direct control — on which community resilience depends. Communities rely on external **duty bearers**; resilience cannot be built by community action alone.

<a id="field-use"></a>
## Field Use

When a community-level project underperforms, check the enabling environment before attributing failure to the community: *"is it because of external policies we can't control?"* Use it to justify advocacy and multi-level engagement alongside community work.

<a id="risks"></a>
## Risks

Holding communities accountable for outcomes determined by higher-level factors; ignoring duty-bearer obligations.
