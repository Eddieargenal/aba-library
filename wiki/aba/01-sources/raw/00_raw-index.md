# Raw Sources Index

## Section Description

- What: Immutable source evidence files (for example PDFs) used as the factual floor of the library.
- Why: Preserves provenance and enables verification against original documents.
- When: Use for citation checks, contested claims, and quality assurance.
- How: Add curated source files only; never modify source contents in place.
- Whom: Source curators, reviewers, and evidence-validation agents.
