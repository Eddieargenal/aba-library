# Summary

This IFRC guide provides Red Cross and Red Crescent National Societies with a framework for engaging in urban resilience programming, covering urban context analysis, resilience programming design, and partnership approaches. It offers practical guidance on designing resilience actions that address urban-specific vulnerabilities including informality, density, governance gaps, and multi-hazard exposure. The guide is relevant to detailed technical design of area-based programs with resilience objectives.
