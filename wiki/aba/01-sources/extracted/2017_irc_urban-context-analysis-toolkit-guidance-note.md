# Summary

The IRC Urban Context Analysis Toolkit provides humanitarian practitioners with operational guidance and instruments for analyzing urban contexts prior to program design. It covers institutional and stakeholder mapping, power analysis, service functionality assessment, market and service provider mapping, and referral pathway development. This toolkit is among the most practically detailed sources in the wiki and provides direct evidence for multiple field instruments and tool pages.
