# Area-Based Reassessment Tool

## Purpose

Provides a structured periodic reassessment protocol for ongoing ABA programmes. Determines whether initial area selection, targeting, and intervention design remain appropriate given evolving conditions — including population movement, service recovery, and changes in local governance.
