# Decision Protocols Index (v2.7)

## Section Description

- What: Decision protocols (`D-`) with conditional logic for high-stakes choices.
- Why: Reduces ad hoc judgment drift and improves auditability.
- When: Use when a recommendation requires explicit if/then criteria.
- How: Define triggers, thresholds, branching logic, and escalation conditions.
- Whom: Decision owners, reviewers, and protocol execution agents.
