# How to Use This Wiki

## For AI agents
1. Read [[index.md]] first for navigation
2. Read [[CLAUDE.md]] for operating rules
3. For any domain question: identify the lifecycle stage, then read the relevant tool pages and source pages
4. For tool use: always read the linked field instruments
5. Never answer without checking source foundation

## For human users
1. Start at [[./00overview/urban-drr-aba-knowledge-map.md]]
2. For a specific field situation: identify your lifecycle stage in [[./06lifecycle/]]
3. Use the tools in [[./04tools/]] with the field instruments in [[./05field-instruments/]]
4. For sector-specific questions: see [[./07sector-applications/]]
5. For coordination issues: see [[./08coordination/]]

## Quality expectations
- Tool pages specify how to collect and analyze evidence, not just what questions to ask
- Source pages link claims to primary documents
- Stubs with TODO markers indicate areas needing further development
