# Query Wiki

## When to use
When answering a domain question about urban DRR, ABA, field tools, or response design.

## Steps
1. Read index.md to identify relevant pages
2. Identify the lifecycle stage relevant to the question
3. Read the relevant tool pages in ./04tools/
4. Read the relevant concept pages in ./02concepts/
5. Read the linked source pages for citation support
6. Read the relevant field instruments in ./05field-instruments/
7. Check ./12risks-contradictions/ for known tensions
8. Produce a practical answer that specifies:
   - Decision question
   - Evidence needed
   - Field data points and collection methods
   - Relevant field instruments
   - Analysis method
   - Decision threshold
   - Source citations
9. If the answer produces reusable synthesis, file it back into the wiki
10. Append query to log.md

## Rule
Never answer a tool question with only conceptual advice. Always include evidence collection and analysis instructions.
