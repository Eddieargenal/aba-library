# Create Decision Memo

## When to use
After completing a tool assessment to document the decision and its evidence base.

## Template
Use [[./05field-instruments/decision-memo-template.md]]

## Steps
1. State the decision question
2. Summarize evidence collected (by domain)
3. Present scoring (if applicable)
4. State the decision recommendation
5. List conditions and caveats
6. List next steps
7. List evidence gaps
8. List dissenting views or unresolved tensions
9. Record date, team, and approvals
10. File in outputs/decision-memos/
11. Append to log.md
