# Tool 12: Implementation Sequencing Dependency Map

## Purpose
A dependency map that sequences program activities in logical order, identifying which activities must precede others, what external dependencies exist (e.g. municipal permits, rainy season), and what parallel workstreams are possible to maintain momentum.

## Decision questions
- What must happen before shelter construction can begin (site clearance, tenure, materials)?
- Can WASH and shelter activities be implemented in parallel or are there dependencies?
- What seasonal or political factors affect implementation sequencing?
