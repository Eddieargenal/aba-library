# Frameworks Index (v2.7)

## Section Description

- What: Decision and methodology frameworks that structure operational analysis.
- Why: Provides repeatable logic for classifying problems and choosing interventions.
- When: Use when scoping assessments, selecting strategy paths, or comparing options.
- How: Link frameworks to concepts, risks, and source-supported constraints.
- Whom: Technical leads, planners, and framework retrieval/synthesis agents.
