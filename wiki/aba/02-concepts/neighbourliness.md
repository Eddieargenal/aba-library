<a id="definition"></a>
## Definition

Neighbourliness (friendliness, helpfulness, respect for privacy; deprivation does not create a "bad" neighbourhood where social cohesion prevails) is a valid proxy indicator for social cohesion and collective action capacity in neighbourhood interventions.

<a id="field-use"></a>
## Field Use

Field query this should answer: *"How do I measure social cohesion progress in a neighbourhood intervention without running a formal survey?"*
