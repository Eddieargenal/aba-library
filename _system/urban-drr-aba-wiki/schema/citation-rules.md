# Citation Rules

## When citing a source in a wiki page
- Link to the source page: [[wiki/01-sources/canonical-source-name]]
- Include author/org and year inline: (Sanderson & Sitko, 2017)
- For direct quotes, note the page number if available

## When a claim is based on multiple sources
- List all source links
- Note if sources agree or have minor differences
- Flag contradictions in wiki/12-risks-contradictions/known-contradictions.md

## Confidence levels
- high: supported by 2+ independent peer-reviewed or practitioner sources
- medium: supported by 1 credible source with no contradicting evidence
- low: inferred or based on a single weak source

## Minimum evidence for tool pages
Each major domain in a tool must be supported by at least two source types:
1. Direct observation
2. Key informant interviews
3. Community consultation
4. Household mini-survey
5. Service provider data
6. Secondary data/maps
7. Existing coordination data

Flag a domain as "weak evidence" if only one source type supports it.
