# 04 — Tools

17 numbered operational tools for field evidence collection, analysis, and decision-making across the ABA/DRR lifecycle. Tool 01 is fully populated and serves as the quality reference; tools 02–17 have skeletal structures with TODO markers and should not be used for field guidance yet.

| [[01-aba-feasibility-and-necessity-assessment-tool]] | **What:** A 9-module structured assessment tool that scores whether an Area-Based Approach is needed, feasible, and justified, producing a recommendation across 5 categories (Full ABA to Do Not Use). **Why:** The entry gate to the ABA lifecycle — prevents committing resources to ABA design in contexts where sector or household approaches would serve better. **When:** Stage 0 (appropriateness decision), before any ABA program design begins. **How:** Complete all 9 modules, score each domain (0–4), apply weights to reach a total score, then generate a decision memo using the decision-memo-template. |
