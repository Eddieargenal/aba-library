# Generate Field Instrument

## When to use
When a tool requires a data collection form, checklist, interview guide, or matrix.

## Steps
1. Identify the tool this instrument supports
2. Identify the decision domains it must provide evidence for
3. Select the instrument format: form | checklist | guide | matrix | survey | dashboard
4. Design the questions/fields:
   - Each question must produce evidence for a specific decision domain
   - Include enumerator guidance for ambiguous questions
   - Include data quality checks (skip patterns, validation ranges, cross-checks)
5. Write the enumerator introduction and instructions
6. Define data quality checks
7. Define analysis use (how responses are used in the tool)
8. Define ethical safeguards (consent, protection, do-no-harm)
9. Create the field instrument page using frontmatter-schema.md
10. Link to the parent tool page
11. Update index.md
12. Append to log.md
