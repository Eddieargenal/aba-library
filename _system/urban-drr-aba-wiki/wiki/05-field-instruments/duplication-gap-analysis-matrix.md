# Duplication and Gap Analysis Matrix

## Purpose
A matrix for systematically identifying where different actors are providing the same activity to the same population in the same area (duplication) and where needs are not being addressed by any actor (gaps). Feeds directly into coordination decisions and area strategy design.

## When to use
- After completing the stakeholder 5W mapping
- At each coordination meeting cycle during implementation
- Before finalizing area strategy to ensure all needs are covered

## Who fills it
Coordination officer or program manager, based on 5W data; validated at coordination meeting.

## Data sources
- Stakeholder 5W mapping form (completed)
- Cluster/sector 4W dataset
- KII notes from actor interviews

## Matrix structure

| Need/Activity | Geographic Zone | Target Population | Actor A | Actor B | Actor C | Overlap? | Gap? | Notes |
|---|---|---|---|---|---|---|---|---|
| [e.g. Shelter repair] | [zone name] | [IDPs in block 3] | [IRC - 200 HH] | [NRC - 150 HH] | - | YES - same HH? | - | Verify targeting lists |
| [e.g. WASH trucking] | [zone name] | [all residents] | - | - | - | - | YES | No actor covering |

## Analysis use
1. Flag all rows with Overlap=YES for coordination follow-up
2. Flag all rows with Gap=YES for area strategy inclusion
3. Prioritize gaps by severity (cross-reference with prioritization matrix)
4. Assign gap-filling responsibility at coordination meeting
