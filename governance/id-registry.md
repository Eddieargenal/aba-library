# ID Registry

## Stable Prefixes

- `S-` source / extracted source
- `C-` concept
- `F-` framework
- `T-` tool
- `I-` field instrument
- `R-` risk
- `KTN-` known tension
- `P-` advisory playbook
- `D-` decision protocol
- `O-` output template / output
- `EP-` evidence packet
- `PU-` proposed update
- `SS-` slice spec

## Registry Rule

Relationships must target stable IDs, never file paths.
